# 项目结构

```text
/github
    ├── README.md               # 项目介绍
    ├─── build-MySQL.md         # 构建项目MySQL表
    ├─── web                    # 前端
    ├─── src
    │   ├─── main
    |   |   ├─── java
    |   |   |   └─── com.example.wiki
    |   |   |       └─── wikiApplication.java       # 启动类
    |   |   └─── resources
    |   |       ├─── static
    |   |       ├─── banner.txt     # Spring启动去除bug
    |   |       └─── application.properties       # 配置信息
    |   └─── target
    └─── pom.xml       # 环境
```
